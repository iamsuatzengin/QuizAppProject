# QuizAppProject

## Kullanılan teknolojiler:
* MVVM - ViewModel, LiveData
* Jetpack Navigation 
* Room - Veritabanı
* Coroutines
* Flow - StateFlow
* Use Cases
* Shared Preferences
* RecyclerView - DiffUtil
* Dagger-Hilt - Dependency Injection
* Lottie - Animasyonlar
* Expandable FAB
